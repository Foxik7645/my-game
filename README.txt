MAP-GAME — минимальный прототип
================================

Быстрый старт (Windows/macOS/Linux)
-----------------------------------
1) Установи Node.js (https://nodejs.org).
2) Распакуй этот архив.
3) Открой терминал в папке проекта.
4) Выполни:
   npm install
   npm start
5) Открой в браузере: http://localhost:3000

Что уже работает
----------------
- Карта OpenStreetMap (Leaflet).
- Клик по карте ставит здание (тип выбирается в селекте).
- Все здания сохраняются в db.json.
- В реальном времени рассылается событие всем игрокам (Socket.IO).

Структура
---------
- server.js       — сервер Express + Socket.IO, REST API, сохранение в файл db.json
- package.json    — скрипты и зависимости
- db.json         — простая "база" (файл)
- public/index.html — клиент: карта, клики, загрузка/отправка построек
